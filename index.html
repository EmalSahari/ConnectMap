Welcome to ConnectMap!

Here's how to use the website:

  1. Add Person: Click this button to add a person to your map. Youâ€™ll be prompted to enter their name.
  2. Add Context: Use this button to add contexts such as "School", "Work", etc., which represent shared places or events.
  3. Add Connection: Select two nodes (persons or contexts) on the map and click this button to connect them. You can also choose to add an indirect connection by specifying an intermediary.
  4. Delete Selected: Select nodes or connections and click this button to remove them.
  5. Customize Colors: Click this button after selecting nodes, edges, or nothing (for background) to change their colors.
  6. Save Map: Save the current state of your map as a JSON file for later use.
  7. Load Map: Load a previously saved JSON file to restore your map.
  8. Download Map: Save the map as a PNG image to share or keep as a reference.

Tips:
- Use Ctrl+Click (or Cmd+Click on Mac) to select multiple items on the map.
- Zoom in/out or drag the map using your mouse.
